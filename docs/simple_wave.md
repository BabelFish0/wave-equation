# Simple wave equation simulation

## Maths

$$
y(x_i, t_{j+1}) = \frac{(\Delta t)^2v^2}{(\Delta x)^2}\left[y(x_{i+1}, t_j)-2y(x_i, t_j)+y(x_i, t_{j+1})\right]-y(x_i, t_{j+1})+2y(x_i, t_j)
$$
